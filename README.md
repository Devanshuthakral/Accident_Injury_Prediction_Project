# 🚗 Accident Injury Prediction using Machine Learning

## 📌 Overview

The **Accident Injury Prediction** project is a Machine Learning application that predicts the severity of injuries in road accidents based on various accident-related factors. By analyzing historical accident data, the model helps identify patterns that contribute to injury outcomes and supports data-driven decision-making for road safety.

---

## 🚀 Features

- Data preprocessing and cleaning
- Handling missing values
- Label Encoding for categorical features
- Exploratory Data Analysis (EDA)
- Train-Test Split
- Multiple Machine Learning models
- Model performance comparison
- Confusion Matrix and Classification Report
- Prediction of accident injury severity

---

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 🤖 Machine Learning Models

The following models are implemented and compared:

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/devanshuthakral/Accident_injury_Prediction_Project.git
```

### Navigate to the project directory

```bash
cd Accident_Injury_Prediction
```

### Install the required dependencies

```bash
pip install -r requirements.txt
```

### Run the project

```bash
jupyter notebook
```

Open the notebook and execute all cells.
